# Charts Overview

For your convenience, you can choose from 10+ commonly used drop-in charts, or create your own data queries for [bar charts](/BarChart), 
[column charts](/ColumnChart), [geo charts](/GeoChart), [line charts](/LineChart), [pie charts](/PieChart), and [tables](/TableChart).

To use a chart, render it with an [AnalyticsDashboard](/AnalyticsDashboard), which handles loading 
the [Google Analytics Embed API](https://devboldly.github.io/react-use-analytics-api/), 
[sign in](/AuthorizeButton) and [sign out](/SignOutButton), and [view selection](/ViewSelector). You can also embed [standalone charts](/custom-dashboards-standalone-charts#standalone-charts).

## Common Charts

Choose from these pre-made charts for common queries:

- [ActiveUsersChart](/ActiveUsersChart) - Visualize userbase growth over time.
- [BounceRateChart](/BounceRateChart) - See how well a site grabs people's attention.
- [OrganicSearchesChart](/OrganicSearchesChart) - Visualize traffic from organic searches.
- [PageViewsPerPathChart](/PageViewsPerPathChart) - See which pages and posts are most popular.
- [PagesPerSessionChart](/PagesPerSessionChart) - See how engaged users are.
- [SessionDurationChart](/SessionDurationChart) - Visualize how long users stay before leaving.
- [SessionsByDateChart](/SessionsByDateChart) - See how traffic grows over time.
- [SessionsByDeviceCategoryChart](/SessionsByDeviceCategoryChart) - See how users consume content.
- [SessionsByHourChart](/SessionsByHourChart) - Know the best time to post content.
- [SessionsBySourceChart](/SessionsBySourceChart) - See which sites traffic comes from.
- [SessionsByUserTypeChart](/SessionsByUserTypeChart) - See the proportion of new and returning visitors.
- [SessionsGeoChart](/SessionsGeoChart) - Visualize where in the world visitors come from.

See each chart for a demo/example, information on usage, and the data queried by each.

### DataChart

A [DataChart](/DataChart) is the base chart used by all of the above charts. It requires a 
query and chart configuration, and uses the [Google Analytics Embed API](https://devboldly.github.io/react-use-analytics-api/) to inject 
the rendered chart into a container on the page.

Each of the custom chart types above wraps [DataChart](/DataChart) and handles configuring the 
chart prop of [DataChart](/DataChart) for you.

You probably won't need to use a DataChart directly since all chart types are available as their
own components above, but you will definitely want to refer to the Query Prop section of 
the [DataChart documentation](/DataChart#query-prop) for creating queries for any custom 
charts you want to build.
